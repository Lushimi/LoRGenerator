Legends of Runeterra Deck Generator

by Studio Gaffers

Specify different genres and generate a random deck from those genres! Made for the Legends of Runeterra card game.

What's a genre? A genre is just something used to bias the randomness of the deck; for example, if I want a deck with many "ephemeral" cards, more cards in the first region, and low cost unit cards. The genres would be keywords, regions, cost, and type!



Import an unfinished deck and work from there, or create a whole deck from scratch!


The code supports both of those things and many different genres, but we don't know how to front end and accept user input so this will just randomly generate 10 decks with random genres.
